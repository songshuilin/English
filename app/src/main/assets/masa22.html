第二节 句子的简化结构

上面我们讲了全球语言句子通用的6大句子成分,但在实际的语言运用中,人们并不是说每一句话时都含有6大句子成分的,那样说话会很啰唆,人们说话时往往会省略一些成分。出现各种各样“短小”的句子。

比如省略宾语后的句子如下：

我们 工作。 我们 走。 我们 停下来。

他们 哭。 他们 跑。 他们 站起来。

火车 到达。 我们 等待。 我 试试。

上面这些句子里第一部分是主语，第二部分是谓语。省略了宾语的句子同样可以带有补语，例如：

他 长 高了。 他 感觉 很冷。 我 感觉 很热。

我 来了 三次。 我 去 就是了。

上面这几个句子里第一部分是主语，第二部分是谓语，第三部分是补语。

还有一种句子，它们的谓语是“是”的意思（此时的谓语又叫“联系动词”)，此时的宾语又叫“表语”，表语除了是名词外，还可以是形容词，我们把这样的句子叫“主语解 + 联系动词 + 表语”句子，或更简单地叫做“主系表”句子，比如：

我 是 一个兵。 我 是 一个老师。

我 是 来自北京的。 他 是 来自上海的。

他 是 兴奋的。 他 是 多才多艺的。

这 是 一个桌子。 这 是 正确的。

那边 是 体育场。 这边 是 足球场。

联系动词，简称为“系动词”，它把两边的东西连接起来。表语，即“表述”的意思，即对主语进行表达和描述。主系表实际上是一种特殊的主谓宾句子。

汉语里不太爱用“是”这个字，因此在汉语里常见到省略了“是”的主系表句子。比如：

我25岁。 （实际上应该说成“我是25 岁”) 我很高兴。 （实际上应该说成“我是很高兴的）

他很重。 （实际上应该说成“他是很重的”) 这本书很难。（实际上应该说成“这本书是很难的”)

他们很富有。（实际上应该说成“他们是很富有的”)
